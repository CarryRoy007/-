## ⚔️ 你，就是战国。

这不是一个让你按部就班"通关"的游戏。没有攻略，没有最优解。

你是秦孝公，面对商鞅递上的变法奏章——**用，则国力暴涨但贵族必反；不用，则错失千古良机**。你是赵王，长平前线廉颇坚守三年，粮草将尽——**换赵括上去赌一把，还是咬紧牙关继续熬？**

每个选择都是一道伤口。选了 A 就永远不知道 B 的结局。选了战，文臣离心；选了和，武将寒心。**每一局都是一段独一无二的历史，每一段历史都以你的谥号盖棺定论。**

---

## 🔥 为什么值得玩

- **6 个纪元，7 个国家，自由开局** — 想从三家分晋白手起家？想在长平之战前夕力挽狂澜？选。
- **没有标准答案的选择题** — 高风险高回报，低风险保平安。是激进还是保守，全在你一念之间。
- **你的每一个决定都在地图上留痕** — 80+ 城池，攻下就变色。看你的版图是扩张还是萎缩。
- **君主会死** — 被刺杀、被毒害、被掳走、积劳成疾……选错了，你的统治戛然而止。然后你选继承人继续。
- **死了也不白死** — 谥号、太史公评价、五星历史定位、后世影响分析。你的每一局都有一个"史官"为你盖棺论定。
- **AI 比你想的活跃** — 各国按性格自动外交、宣战、结盟。你不打别人，别人会来打你。

---

## 🎮 怎么玩

1. 选纪元 → 选国家 → 开始
2. 每回合弹出事件，读半文言描述，用你的判断选 A/B/C/D
3. 看地图上你的城池变色，看国力栏的数字跳动
4. 死了就选继承人继续，或看完谥号评价后重开
